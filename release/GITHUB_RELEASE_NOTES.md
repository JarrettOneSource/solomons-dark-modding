# Solomon Dark Multiplayer Beta v0.1.0-beta.11

- Client-owned gold, orbs, items, every potion subtype, miscellaneous drops, and powerups now replay their stock popup, sound, particles, and retirement exactly once after host authorization.
- Earth Boulder and continuous Water Frost Jet casts retain their native visuals on remote peers.
- Boneyard generation, trees, and scrub now follow the host's stock generation boundary.
- Dead players receive the stock death presentation for about three seconds, then spectate named living players; either mouse button cycles targets and the next wave respawns everyone at the spawn point.
- The launcher keeps its own local save slots, can import and select saves, and performs local-first cloud backups only after the active Steam account is linked on the website.
- Host Game no longer opens the Steam invite picker. Host and Join enter the character loadout flow directly, and website lobby links route into the launcher.
- Launcher and website-installed mod updates are transactional and require confirmation where appropriate.
- The release contains no bundled mods or generated runtime residue.

Download the ZIP. Extract the ZIP. Start `SolomonDarkMultiplayerBeta.exe`.
