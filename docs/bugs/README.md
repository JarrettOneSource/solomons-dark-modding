# Bugs

This folder is for documenting game bugs that matter to the mod loader project,
especially pre-existing stock-client bugs that the loader can expose, log,
reproduce, or work around.

Use it to capture things like:

- what the bug is
- whether it appears to be stock-client behavior or loader-induced behavior
- how to reproduce it
- what evidence we have
- what the current status is
- what the next investigation or fix path should be

Prefer one focused markdown file per bug or bug family.

Keep the notes concrete. Favor reproduction steps, observed behavior, exception
details, logs, and current interpretation over vague summaries.

If a bug is eventually fixed, keep the original write-up and update its status
instead of deleting the history.
